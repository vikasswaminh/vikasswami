20 Days • 20 Technologies Internship Challenge

Hands-on immersion into virtualization, firewalls, VPNs, routing, monitoring, automation, and HA — designed around real global infrastructure pods located in Paris.

Benefits of the 20-Day • 20-Technologies Internship

A complete immersion in enterprise-level operations — from datacenter virtualization to SOC monitoring — equipping students to handle live environments confidently.

Practical Exposure to Cutting-Edge Tech

  • Virtualization: VMware ESXi & Proxmox
  • Firewalls: pfSense and FortiGate
  • Routing: Cisco IOS, MikroTik OSPF
  • VPNs: WireGuard, OpenVPN, Tailscale
  • Monitoring: Zabbix, Grafana, Prometheus, Suricata
  • Automation: Ansible, Docker
  • HA / Backup: Proxmox Cluster + DR

Enterprise-Grade Learning Outcomes

  • Design and deploy multi-site network topologies.
  • Build and manage secure firewall / routing policies.
  • Perform live monitoring and incident analysis in NOC / SOC style.
  • Automate configurations using modern IaC tools.
  • Manage VM migrations and snapshots across hypervisors.
  • Implement backup and high-availability strategies.

Tool Access & Infrastructure

  • Controlled access to licensed or trial enterprise software.
  • Centralized logging and telemetry like real NOC/SOC suites.
  • Private lab pods hosted in Paris Datacenter.
  • Secure RDP/SSH/VPN access via Tailscale mesh.

Remote Management & Collaboration

  • Pods connected through a central management fabric.
  • Simulated multinational NOC/SOC workflow environment.
  • Mentor dashboards for uptime and performance monitoring.
  • Live supervision and ticket escalation channels.

Career Advantage

  • Hands-on portfolio in 20 enterprise technologies.
  • Industry-ready skills for NOC, SOC, DevOps & Cloud roles.
  • Confidence handling global hybrid infrastructure.
  • Mentor-verified certification on completion.
